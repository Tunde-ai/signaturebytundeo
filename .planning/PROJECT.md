# Signature By Tunde O

## Vision
Digital platform for Signature By Tunde O — a wealth strategy and notary services business. Combines public marketing site with FinClear client-accountant collaboration portal, AI wealth advisor chatbot, and consultation booking.

## Stack
- Next.js 16 + React 19 + TypeScript 5
- Tailwind CSS 4 (luxury navy/gold theme)
- NextAuth.js 5 (JWT, credentials provider)
- Anthropic Claude API (AI chatbot + financial reports)
- Resend (email service)
- Vercel deployment

## What's Built
- Home/Hero with services overview
- Wealth Waterfall strategy explainer (4-tier system)
- Notary services page (Commission #HH389868)
- Contact form with Resend integration
- FinClear login portal (accountant + demo client)
- FinClear app: Dashboard, Ledger, Vault, AI Reports, Notes
- AI wealth advisor chatbot (Claude-powered)
- Role-based views (accountant vs client)
- 24 demo transactions, full collaboration features

## Key Details
- Brand: Playfair Display + Inter fonts
- Colors: Navy #0B1221, Gold #C9A84C, Cream #F5F0E8
- Deployed on Vercel
- Demo data (no database integration yet)

## Status
MVP complete and deployed. FinClear portal uses demo data — needs database integration for production use.
